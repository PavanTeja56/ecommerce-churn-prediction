# Feature Dictionary

## Core Features (RFM)

- Recency: Days since last purchase
- Frequency: Number of unique invoices
- Monetary: Total revenue
- TotalQuantity: Total items purchased
- AvgOrderValue: Average revenue per invoice

## Behavioral Features

- TotalInvoices
- CustomerLifetimeDays
- AvgDaysBetweenPurchases
- PurchaseRatePerMonth

## Temporal Features

- WeekendPurchaseRatio
- EveningPurchaseRatio
- UniqueActiveMonths
- UniqueActiveDays
- MonthlyPurchaseVariance

## Product Diversity Features

- UniqueProducts
- ProductDiversityRatio
- AvgItemsPerInvoice
- RepeatPurchaseRatio
- TopProductConcentration

## Engagement Features

- RevenuePerMonth
- QuantityPerMonth
- InvoiceFrequencyScore
- RecencyScore
- MonetaryScore
- EngagementScore
